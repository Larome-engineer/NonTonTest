# При разработке использовались:
> JDK: Oracle OpenJDK 17
> SDK: 17
> Стандартные библиотеки Java

# Архитектура: MVC-архитектура (* С примечаниями)

* Примечания:
----------------------------------------------------------------------------------
1. Вместо View использую понимание UI для, в котором реализовал консольный интерфейс

2. Controller не использую, поскольку, в данной ситуации это незачем

3. Repository реализую в виде паттерна DAO в пакете dao. Вместо БД используется HashMap
    3.1 В ProductDao происходит проверка на существование продукта и CRUD операции с HashMap

4. ProductService - сервис, в котором описывается логика взаимодействия с данными Product

5. Model - наш класс Product, который имеел поля: String id и String name

6. ProductUi - UI для взаимодействия пользователя с приложение через консоль
